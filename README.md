## Daftar Isi Materi
Setiap materi diimplementasikan dalam folder terpisah dan siap dijalankan secara lokal (menggunakan Live Server).

| No. | Folder | Materi Inti | Penjelasan Singkat |
| :--- | :--- | :--- | :--- |
| 1 | `01-Fungsi-JS` | **Function** | Program CRUD sederhana untuk demonstrasi fungsi (Create, Read, Update, Delete). |
| 2 | `02-Objek-JS` | **Object** | Implementasi Object Literal dengan properti, method (`this`), dan manipulasi objek. Hasil ditampilkan di **Console Browser (F12)**. |
| 3 | `03-DOM-JS` | **DOM Manipulation** | Modifikasi konten, *style*, dan struktur HTML (DOM) menggunakan JavaScript. |
| 4 | `04-AJAX` | **AJAX (XHR)** | Memuat data secara asinkron (Async JavaScript) menggunakan objek `XMLHttpRequest` untuk mengambil data dari `data.json`. |
| 5 | `05-Event-JS` | **Event Handling** | Demonstrasi penanganan Event, termasuk `preventDefault()` dan Event Propagation (Bubbling & Capturing). |

## Cara Menjalankan

1.  **Clone Repositori:** Unduh semua file ke komputer lokal Anda.
    ```bash
    git clone [https://github.com/najwaandistysa/Tugas-JS-RPL-Final.git](https://github.com/najwaandistysa/tugas-js.git)
    ```
2.  **Buka di VS Code:** Buka folder repositori di Visual Studio Code.
3.  **Jalankan:** Buka file `.html` yang diinginkan di setiap folder menggunakan *Live Server* atau langsung di *browser*.
